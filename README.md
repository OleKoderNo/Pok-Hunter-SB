# PokéHunter SB

A lightweight, persistent Pokémon collection game for **Streamer.bot** using **Twitch Channel Point Rewards**.

Viewers redeem Channel Points to encounter Pokémon, collect shiny Pokémon, and slowly build their own permanent Pokédex. Every catch is automatically saved, allowing viewers to continue growing their collection across multiple streams.

PokéHunter SB is designed to be simple to set up, easy to customize, and lightweight enough to run entirely from Streamer.bot without requiring a database or additional software.

# ✨ Features

- 🎮 One Pokémon encounter per Channel Point redemption
- 💾 Permanent collections saved for every viewer
- ✨ Shiny Pokémon (default 1 in 250 chance)
- 🌍 Regional Pokémon forms
- 🛡 Duplicate protection
- 📈 Loss streak protection with additional encounter attempts
- 🎲 Weighted Pokémon encounters based on BST, rarity and typing
- 🔄 Automatic save file creation
- 🗂 Automatic backups
- ⚡ No database required
- 🌐 Works completely offline after the initial setup
